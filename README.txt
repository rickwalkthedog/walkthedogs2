Walk The Dogs static website for Cloudflare Pages

Files included:
- index.html
- services.html
- about.html
- scheduling.html
- contact.html
- styles.css
- assets folder with images

How to publish on Cloudflare Pages:
1. Log in to Cloudflare.
2. Go to Workers & Pages.
3. Create Application > Pages.
4. Choose Direct Upload, or connect a GitHub repository.
5. Upload this entire folder.
6. Add the custom domain: walkthedogs.net.
7. Cloudflare will automatically provide SSL.

Scheduling:
Create your services in Square Appointments or Calendly, then paste their embed code into scheduling.html where the booking box is.

Contact form:
The current contact form is static. To receive email submissions, connect a form service such as Formspree/Basin or use Cloudflare Workers. The phone/text links work immediately.
