## Cloudflare Pages Settings

Use these settings:

```text
Framework preset: None
Build command: leave blank
Build output directory: /
```

## Custom Domain

After deployment, go to your Pages project and add:

```text
walkthedogs.net
www.walkthedogs.net
```

Cloudflare will issue SSL automatically.

## Calendly

The scheduling page embeds:

```text
https://calendly.com/walk-the-dogs
```

## Contact Form

The home page uses StaticForms.dev with the access key supplied by the site owner.
